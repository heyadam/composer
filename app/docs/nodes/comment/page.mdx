# Comment

Add notes and visually organize your flow.

## Overview

Comment nodes are resizable boxes for adding annotations, grouping related nodes, or documenting your workflow.

## Features

### Editable Content

- **Title** - A heading for the comment
- **Description** - Detailed notes or explanation

### Color Themes

Choose from multiple color themes to visually distinguish different sections:
- Gray (default)
- Blue
- Green
- Yellow
- Purple
- Pink
- Orange

### AI Suggestions

Comment nodes can automatically generate title and description based on nearby nodes. This helps document your flow with minimal effort.

## Grouping Nodes

Drag other nodes onto a comment to group them. Grouped nodes:
- Move together when you drag the comment
- Stay organized visually
- Can be ungrouped by dragging them out

## Tips

- Use comments to label sections of complex flows
- Color-code different stages of your pipeline
- Add documentation for collaborators
- Group related nodes for easier navigation
